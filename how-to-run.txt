# How to Run the IM App (Signal Protocol E2EE)

## 1. Backend Setup (FastAPI)
Open a terminal and run the following commands:

cd backend

# Create a virtual environment
python -m venv venv

# Activate the virtual environment
# For Windows:
venv\Scripts\Activate
# (If you get an error, run: Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser, then try again)
# For Mac/Linux:
# source venv/bin/activate

# Install dependencies
pip install -r requirements.txt

# Run the server (must be inside the activated venv)
uvicorn main:app --reload
# The backend will run on http://localhost:8000


## 2. Frontend Setup (React)
Open a SECOND, separate terminal and run:

cd frontend

# Prerequisites: Ensure you have Node.js installed (v20+)
# Install all required packages (React, Tailwind, Lucide, Axios, etc.)
npm install

# Start the development server
npm run dev
